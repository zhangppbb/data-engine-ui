企业工商基本信息查询

新增

接口标识: get_company_basic_info

根据企业名称/统一社会信用代码查询企业工商注册基本信息，包含法人、注册资本、注册地址、成立日期等核心工商数据

工商登记 基础数据 实时

请求参数

参数名 类型 必填 描述
company_name string ✓ 企业名称
credit_code string ✗ 统一社会信用代码
detail boolean ✗ 是否返回详细信息

请求示例

POST /api/v1/company/basic
Content-Type: application/json

{
  "company_name": "阿里巴巴集团",
  "credit_code": "91330100716157688L",
  "detail": true
}

成功响应

{
  "code": 200,
  "message": "success",
  "data": {
    "company_name": "阿里巴巴集团控股有限公司",
    "credit_code": "91330100716157688L",
    "legal_person": "黄勇",
    "registered_capital": "5000000000",
    "registered_address": "浙江省杭州市余杭区文一西路969号",
    "established_date": "1999-09-09",
    "business_scope": "技术开发、技术服务、技术咨询、技术转让",
    "status": "存续",
    "industry": "互联网和相关服务",
    "employees": 120000
  }
}

失败响应

{
  "code": 404,
  "message": "企业信息未找到",
  "data": null
}

错误码说明

错误码 描述 解决方案
400 参数错误 检查请求参数格式
404 企业信息未找到 检查企业名称或信用代码是否正确
403 权限不足 检查API密钥和权限
500 服务器内部错误 稍后重试或联系技术支持

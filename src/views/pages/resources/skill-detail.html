企业竞品综合分析Skill

针对目标企业与行业竞品进行多维度对比分析，自动生成完整分析报告，包含市场占有率、核心竞争力对比、优劣势分析等内容

新增 AI技能 可复用

请求参数

参数名 类型 必填 描述
company_name string 是 目标企业名称
industry string 否 行业分类
competitors array 否 竞品企业名称列表
analysis_dimensions array 否 分析维度，默认包含市场、产品、技术、财务
output_format string 否 输出格式，可选值：json、markdown、html

  • 财务状况分析
  • 发展趋势预测

详细报告示例

企业竞品综合分析报告

1. 市场占有率分析
某科技有限公司在人工智能领域的市场占有率为35%，位居行业第一。竞品公司A和B分别为28%和22%。

2. 核心竞争力对比
某科技有限公司在技术创新和市场拓展方面具有明显优势，但在品牌知名度和海外市场方面仍有提升空间。

3. 战略建议
建议加强品牌建设，提升品牌知名度；同时积极拓展海外市场，特别是东南亚和欧洲市场。

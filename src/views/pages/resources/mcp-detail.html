实际控制人

get_actual_controller

查询企业的实际控制人详情，适用于企业风险评估、关联交易识别及商业竞争对手分析场景。

企业基座 实时

请求参数

参数名 类型 必填 描述
searchKey string 是 企业名称或统一社会信用代码
timeout number 否 超时时间（毫秒），默认3000

服务说明

数据来源

  • 国家企业信用信息公示系统
  • 市场监督管理部门官方数据
  • 权威第三方数据源

错误码说明

错误码 描述 解决方案
400 参数错误 检查必填参数是否正确
403 风险熔断 企业存在失信风险，需评估后操作
429 请求频率过高 降低请求频率，遵守API调用限制
500 服务器内部错误 稍后重试或联系技术支持
503 服务暂不可用 稍后重试或查看服务状态
